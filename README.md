## Project Description

Thirukkural Wisdom is an interactive iOS mobile application designed to help users explore the 1330 Kurals of Thirukkural. The app presents each Kural with Tamil text, English meaning, audio playback, images, search, favorites, and easy navigation.

## Main Features

* View Thirukkural verses
* Browse Kurals by chapter
* Read Tamil text and English meaning
* Play audio for each Kural
* View related images
* Search Kurals by number, chapter, or keyword
* Favorites section
* Clean SwiftUI interface
